# My Neovim Config

This is my personal Neovim configuration built on [LazyVim](https://github.com/LazyVim/LazyVim). It's constantly evolving as I tweak things to match my workflow.

## What's in here

- **LazyVim** as the base - gives me a solid LSP setup and sensible defaults
- **Custom colorscheme** (`dankcolors`) - a base16 theme I personally tune. Dark background with pink accents and green strings
- **Terminal QoL** - escape from terminal mode with `<Esc>`, jump between splits with `<C-h/j/k/l>`
- **Rust analyzer** tuned for my projects (all cargo features, some proc macros ignored)
- **Snacks.nvim** - shows hidden files in picker/explorer

## Notes

This isn't a template or a project. It's just my config, flavored the way I like it. Things change here often as I find better ways to do things or discover new plugins.

If you're looking for a stable, well-documented setup, check out [LazyVim](https://lazyvim.github.io). This is just my personal take on it.
